# PRISM Archive

> 웹의 모든 텍스트를 가장 직관적으로 수집하는 방법  
> 드래그 앤 드롭 한 번으로 웹사이트, 기술 문서, 그리고 대화형 AI(ChatGPT, Claude 등)의 중요한 순간을 영구적으로 저장하세요.

---

## 💎 소개 (Introduction)

**PRISM Archive**는 웹 서핑 중 발견한 중요한 텍스트 조각을 드래그 앤 드롭만으로 즉시 저장하고 관리하는 크롬 확장 프로그램입니다.

저장된 항목을 클릭하면 해당 웹페이지로 이동하여 저장했던 문장을 자동으로 찾고 하이라이트 처리합니다.

특히 스크롤이 긴 기술 문서나 AI와의 긴 대화 로그를 저장하는 데 최적화되어 있습니다.

---

## ✨ 주요 기능 (Key Features)

![](https://raw.githubusercontent.com/ammbush/Prism_Archive/refs/heads/main/images/%EC%BA%A1%EC%B3%902.png)

### 1️⃣ 직관적인 수집 (Drag & Drop Archiving)

- 웹페이지에서 텍스트를 드래그하면 우측 하단에 **PRISM Dropzone (❖)** 표시
- 드래그 후 놓기만 하면 저장 완료 (복사/붙여넣기 불필요)
- 모든 사이트에서 작동
- CSS 충돌 방지 처리로 레이아웃 영향 없음

### 2️⃣ 스마트 리콜 & 하이라이팅 (Smart Recall)

- 저장된 카드를 클릭하면 원본 페이지로 이동
- 자동 스크롤 엔진이 저장된 문장을 정확히 탐색
- 시각적으로 하이라이트 표시

### 3️⃣ 강력한 관리 기능 (Management)

- **Timeline View** — 시간순 기록 확인
- **Domain View** — 도메인별 자동 분류
- **Custom Folders** — 사용자 폴더 생성
- **Batch Actions** — 다중 선택 후 이동 / 삭제 / 열기

### 4️⃣ 사용자 편의성 (User Experience)

- 🌙 다크 모드 지원
- 🔍 실시간 검색 (제목 / 내용 / URL)
- 🧩 Favicon 자동 표시
- 💾 로컬 스토리지 저장 (회원가입 없음)

## 📦 설치 방법 (Installation)

이 프로젝트는 크롬 확장 프로그램 개발자 모드를 통해 설치할 수 있습니다.

1.  **다운로드**: 이 저장소의 [Code] 버튼을 눌러 **Download ZIP**을 하거나 `git clone` 합니다.
2.  **압축 해제**: 다운로드한 ZIP 파일의 압축을 풉니다.
3.  **확장 프로그램 관리 페이지 이동**: 크롬 주소창에 `chrome://extensions`를 입력하고 접속합니다.
4.  **개발자 모드 활성화**: 우측 상단에 있는 **'개발자 모드(Developer mode)'** 스위치를 켭니다.
5.  **로드하기**: 좌측 상단 **'압축해제된 확장 프로그램을 로드합니다(Load unpacked)'** 버튼을 클릭합니다.
6.  **폴더 선택**: 압축을 푼 폴더(manifest.json이 들어있는 폴더)를 선택하면 설치가 완료됩니다!

## 📖 사용 가이드 (User Guide)

### 텍스트 저장하기

1. 웹페이지에서 텍스트 드래그
2. 우측 하단 ❖ 아이콘으로 드롭
3. ✓ 표시 후 저장 완료

### 저장 내용 관리

1. 브라우저 퍼즐 아이콘 → PRISM Archive 핀 고정
2. 아이콘 클릭 → 팝업 열기
3. Timeline / Domains / My Folders 확인
4. ✓ 버튼 → 다중 선택 모드
