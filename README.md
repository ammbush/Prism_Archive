# PRISM Archive

> 웹의 모든 텍스트를 가장 직관적으로 수집하는 방법  
> 드래그 앤 드롭 한 번으로 웹사이트, 기술 문서, 그리고 대화형 AI(ChatGPT, Claude 등)의 중요한 순간을 영구적으로 저장하세요.

---

## 💎 소개 (Introduction)

**PRISM Archive**는 웹 서핑 중 발견한 중요한 텍스트 조각을 드래그 앤 드롭만으로 즉시 저장하고 체계적으로 관리하는 크롬 확장 프로그램입니다.

단순히 링크만 저장하는 것이 아니라, **선택한 텍스트의 문맥(Context)**을 함께 저장합니다.  
저장된 항목을 클릭하면 해당 웹페이지로 이동하여 저장했던 문장을 자동으로 찾고 하이라이트 처리합니다.

특히 스크롤이 긴 기술 문서나 AI와의 긴 대화 로그를 저장하는 데 최적화되어 있습니다.

---

## ✨ 주요 기능 (Key Features)

### 1️⃣ 직관적인 수집 (Drag & Drop Archiving)

- 웹페이지에서 텍스트를 드래그하면 우측 하단에 **PRISM Dropzone (❖)** 표시
- 드래그 후 놓기만 하면 저장 완료 (복사/붙여넣기 불필요)
- 모든 사이트에서 작동
- CSS 충돌 방지 처리로 레이아웃 영향 없음

### 2️⃣ 스마트 리콜 & 하이라이팅 (Smart Recall)

- 저장된 카드를 클릭하면 원본 페이지로 이동
- 자동 스크롤 엔진이 저장된 문장을 정확히 탐색
- 시각적으로 하이라이트 표시

### 3️⃣ 강력한 관리 기능 (Management)

- **Timeline View** — 시간순 기록 확인
- **Domain View** — 도메인별 자동 분류
- **Custom Folders** — 사용자 폴더 생성
- **Batch Actions** — 다중 선택 후 이동 / 삭제 / 열기

### 4️⃣ 사용자 편의성 (User Experience)

- 🌙 다크 모드 지원
- 🔍 실시간 검색 (제목 / 내용 / URL)
- 🧩 Favicon 자동 표시
- 💾 로컬 스토리지 저장 (회원가입 없음)

## 📖 사용 가이드 (User Guide)

### 텍스트 저장하기

1. 웹페이지에서 텍스트 드래그
2. 우측 하단 ❖ 아이콘으로 드롭
3. ✓ 표시 후 저장 완료

### 저장 내용 관리

1. 브라우저 퍼즐 아이콘 → PRISM Archive 핀 고정
2. 아이콘 클릭 → 팝업 열기
3. Timeline / Domains / My Folders 확인
4. ✓ 버튼 → 다중 선택 모드
